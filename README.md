genregen-sinatra
================

A Sinatra frontend for [genregen](https://github.com/and-the-rest/genregen).

## Usage

### Installation

```bash
$ git clone git@github.com:and-the-rest/genregen-sinatra.git
$ cd genregen-sinatra
$ bundle
```

### Running

```bash
$ ruby app.rb # takes normal sinatra arguments
```
